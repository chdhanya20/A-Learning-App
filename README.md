# Skillify - Learning Management System

->A full-stack Learning Management System (LMS) web application built using modern technologies. This platform allows users to browse courses, enroll, and track learning progress.


Features

* 🔐 User Authentication (Login/Register)
* 📖 View Available Courses
* ✅ Enroll in Courses
* 📊 Track Enrolled Courses
* ⚡ Real-time API Integration
* 🎨 Modern UI with animations
* 📱 Responsive Design

---

🛠️ Tech Stack

### Frontend

* React.js
* CSS / Tailwind (if used)
* Axios

### Backend

* Node.js
* Express.js

### Database

* MySQL / MongoDB (based on your setup)

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```
git clone https://github.com/chdhanya20/A-Learning-App.git
cd A-Learning-App
```

---

### 2️⃣ Setup Backend

```
cd backend
npm install
node server.js
```

---

### 3️⃣ Setup Frontend

```
cd frontend
npm install
npm start
```

---

## 🌐 API Endpoints

* `GET /courses` → Get all courses
* `POST /enroll` → Enroll in course
* `GET /enrolled` → Get enrolled courses
